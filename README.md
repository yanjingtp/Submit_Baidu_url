# Submit_Baidu_url
使用百度API提交站点链接

百度收录中提交站点地图抓取特别慢，只有通过API提交才可快速被收录，但是手动提交效率太低，因此通过解析站点地图，然后使用百度API快速提交站点链接。

如需java版源码,可参考[yanjing的博客](https://blog.yanjingtp.cn/articles/2022/02/16/1644981637458.html)

将源码打包为jar,部署到服务器中,设置定时执行,即可自动提交.
